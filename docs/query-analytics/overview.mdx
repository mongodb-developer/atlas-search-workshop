# Query Analytics Overview

:::note
Lab Goals:
* Understand importance of query analytics
* Add tracking to our `$search` queries
* Add and effectively use syonyms
:::

:::caution
If your application, like this library search interface does, performs searches for practically every keystroke, you 
may want to only add `tracking` on fully formed search requests. Tracking partial queries may or may not be useful or
clutter the insights of Query Analytics.
:::
