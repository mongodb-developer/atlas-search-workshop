<details>
<summary>knnVector index configuration</summary>
<div>

```js
{
  "mappings": {
    "dynamic": true,
    "fields": {
      "v": [
        {
          "dimensions": 2,
          "similarity": "euclidean",
          "type": "knnVector"
        }
      ]
    }
  }
}
```

</div>
</details>

<details>
<summary>$search for nearest neighbors</summary>
<div>

```js
[
  {
    $search: {
      index: "default",
      knnBeta: {
        vector: [0.5, 0.5],
        path: "v",
        k: 2,
      },
    },
  },
  {
    "$project": {
      "score": { $meta: "searchScore" }
    }
  }
]
```
